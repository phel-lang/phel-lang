# Lang Module

## Purpose
Core runtime data types and structures for the Phel language.

## Responsibilities
- Persistent data structures (lists, vectors, maps, sets)
- Language primitives (Keywords, Symbols, Variables)
- Type interfaces and protocols (Equalizer, Hasher, etc.)
- Metadata support for runtime types
- Source location tracking
- Collection interfaces (Push, Remove, Contains, Slice, etc.)
- Function interfaces and abstractions
- Runtime registry and type system foundation
